🍽️ Swiggy Restaurant Analysis Dashboard | Python & Power BI Project.......
📌 Project Overview

This project presents an interactive Power BI dashboard built using a Swiggy restaurant dataset. The objective was to analyze restaurant performance, pricing, customer ratings, and delivery times to generate meaningful business insights through data visualization.

The dashboard enables users to explore restaurant data across different areas using interactive filters and KPI cards, making it easier to identify trends and support data-driven decision-making.

🎯 Objectives
Analyze restaurant distribution across different areas.
Compare restaurant pricing and customer ratings.
Evaluate delivery time performance.
Build an interactive dashboard for business insights.
Practice Power BI, DAX, and dashboard design using a real-world dataset.
